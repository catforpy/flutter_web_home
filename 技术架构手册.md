## 🎯 核心架构概述

### 架构模式
采用 **Clean Architecture（清洁架构）** + **领域驱动设计（DDD）**

### 分层架构
```
┌─────────────────────────────────────────┐
│       Presentation Layer (表现层)        │
│     Pages, Widgets, Theme, Routes        │
└──────────────┬──────────────────────────┘
               ↓
┌─────────────────────────────────────────┐
│      Application Layer (应用层)          │
│           BLoCs, UseCases                │
└──────────────┬──────────────────────────┘
               ↓
┌─────────────────────────────────────────┐
│        Domain Layer (领域层)              │
│      Entities, Repositories              │
└──────────────┬──────────────────────────┘
               ↑
┌──────────────┴──────────────────────────┐
│         Data Layer (数据层)               │
│    Models, DataSources, Mappers          │
└──────────────────────────────────────────┘
```

## 🔄 核心流程

### 路由流程
```
用户点击导航
    ↓
触发路由事件
    ↓
go_router 处理
    ↓
更新 URL
    ↓
渲染新页面
```

### 状态管理流程
```
UI 触发事件
    ↓
发送 Event 到 BLoC
    ↓
BLoC 处理业务逻辑
    ↓
Emit 新的 State
    ↓
UI 更新
```

### 数据请求流程
```
UI 发起请求
    ↓
调用 UseCase
    ↓
调用 Repository
    ↓
调用 DataSource
    ↓
返回数据
    ↓
UI 更新
```

## 🎨 设计模式

### 1. BLoC 模式（状态管理）

**文件**: `lib/application/blocs/`

**示例**:
```dart
// Event
class IncrementEvent extends CounterEvent {}

// State
class CounterState {
  final int count;
}

// BLoC
class CounterBloc extends Bloc<CounterEvent, CounterState> {
  // 处理事件，更新状态
}

// UI 使用
BlocBuilder<CounterBloc, CounterState>(
  builder: (context, state) => Text('${state.count}'),
)
```

### 2. Repository 模式（数据访问）

**接口定义**（Domain层）:
```dart
abstract class ContentRepository {
  Future<Either<Failure, List<Content>>> getContents();
}
```

**实现**（Data层）:
```dart
class ContentRepositoryImpl implements ContentRepository {
  final ContentRemoteDataSource remoteDataSource;
  final ContentLocalDataSource localDataSource;

  @override
  Future<Either<Failure, List<Content>>> getContents() async {
    try {
      final remoteData = await remoteDataSource.getContents();
      return Right(remoteData);
    } catch (e) {
      return Left(ServerFailure());
    }
  }
}
```

### 3. 依赖注入

**使用 MultiBlocProvider**:
```dart
MultiBlocProvider(
  providers: [
    BlocProvider(create: (_) => CounterBloc()),
    BlocProvider(create: (_) => ThemeBloc()),
  ],
  child: MaterialApp.router(...),
)
```

## 🚀 性能优化

### 1. 代码分割
```dart
// 懒加载路由
GoRoute(
  path: '/about',
  builder: (context, state) => const AboutPage(),
)
```

### 2. Const 构造函数
```dart
// ✅ 好
const Text('Hello');

// ❌ 不好
Text('Hello');
```

### 3. ListView.builder
```dart
// ✅ 好
ListView.builder(
  itemCount: items.length,
  itemBuilder: (context, index) => ItemWidget(items[index]),
)

// ❌ 不好
ListView(
  children: items.map((item) => ItemWidget(item)).toList(),
)
```

### 4. 图片优化
```dart
CachedNetworkImage(
```
