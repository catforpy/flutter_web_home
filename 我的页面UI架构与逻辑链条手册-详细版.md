## 页面功能说明

### 1. 用户信息横幅功能
- **位置**: 页面顶部
- **背景**: 深蓝色渐变 (#2C5F8D → #1E3A5F)
- **用户头像**:
  - 显示逻辑: authState.avatarUrl 不为空则显示网络图片，否则显示默认图标
  - 网络头像加载失败时自动切换到默认头像
  - 白色边框 3px
  - 头像尺寸: 100x100
- **用户名**:
  - 显示优先级: realName > username > "未设置"
  - 字体大小: 28px
  - 字体颜色: 白色
- **VIP标签**:
  - 金色背景 (#FFD700)
  - 白色文字
  - 圆角 4px
  - 内边距: 水平12px, 垂直4px
- **数据统计**:
  - 客户: 显示空白
  - 商户: 显示"完成订单"、"服务评分"、"满意度"、"模板数"、"客户数"
  - 后台: 显示"任务"、"完成"、"排名"、"站内信"
  - 未登录: 不显示

### 2. 标签导航功能
- **我的公司** (index: 0):
  - 显示: 项目展厅、我的公司、我的小程序三个模块
  - 项目展厅 (_buildMyAppSection): 展示用户的项目
  - 我的公司 (_buildMyCompanySection): 显示已关联的公司列表，可添加新公司
  - 我的小程序 (_buildMyMiniProgramSection): 显示已注册的小程序列表
- **我在销售** (index: 1):
  - 筛选标签: "全部"、"已上架"、"开发中"
  - 小程序网格: 2列布局，显示小程序卡片
  - 卡片包含: 缩略图、名称、简介、价格、操作按钮
- **我在租赁** (index: 2):
  - 筛选标签: "全部"、"已上架"、"开发中"
  - 租赁服务网格: 显示租赁服务卡片
- **我在合作** (index: 3):
  - 筛选标签: "全部"、"进行中"、"已完成"
  - 合作项目网格: 显示合作项目卡片

### 3. 公司列表功能
- **公司卡片**:
  - 公司图标: Icons.business, 蓝色 #1890FF
  - 公司名称: 加粗显示，黑色 #333333
  - 公司类型和成立时间: 灰色 #666666
  - 状态标签: 绿色 #00C853, "已认证"
  - 点击卡片 → 跳转到公司详情页面
  - 公司详情显示: 基本信息、关联小程序、操作记录

### 4. 添加公司功能
- **表单字段**:
  - 公司名称 (必填)
  - 统一社会信用代码 (必填, 18位数字或大写字母)
  - 法定代表人 (必填)
  - 联系电话 (必填, 手机号格式)
  - 公司地址 (可选)
- **验证规则**:
  - 所有必填字段不能为空
  - 统一社会信用代码必须是18位数字或大写字母
  - 联系电话必须是有效的手机号格式
- **提交后**:
  - 显示成功提示
  - 公司信息保存到后端 (TODO)
  - 刷新公司列表

### 5. 注册小程序功能
- **表单字段**:
  - 小程序名称 (必填)
  - 小程序类型 (必选, 下拉选择)
  - 小程序简介 (必填)
  - Logo (可选, 文件上传)
  - 截图 (可选, 最多5张)
- **小程序类型选项**:
  - 教育培训
  - 电商购物
  - 生活服务
  - 企业展示
  - 餐饮美食
  - 酒店旅游
  - 其他
- **提交后**:
  - 显示成功提示
  - 自动跳转到工作台 (TODO)
  - 小程序信息保存到后端 (TODO)

### 6. 导航栏功能
- **统一导航栏组件**:
  - 首页、购买、租赁、合作、定制开发菜单项
  - 工作台菜单项 (仅对已登录且为特定用户类型的用户显示)
  - 订单按钮
  - 购物车按钮
  - 认证组件 (登录/注册/用户头像)
- **选中高亮**:
  - 当前路径与菜单项路径匹配时显示高亮
  - 选中样式: 红色背景 #D93025, 白色文字
  - 未选中: 透明背景, 黑色文字

### 7. 权限控制
- **工作台访问控制**:
  - 需要已登录
  - 用户类型为以下之一:
    - 客户 (platform_account)
    - 商户 (service_provider)
    - 平台管理员 (platform_admin)
    - 后台管理员 (backend_admin)
- **未登录用户**: 显示登录/注册按钮
- **已登录但非上述类型**: 不显示工作台菜单项

### 8. 数据状态管理
- **_selectedTabIndex**: 当前选中的主标签索引 (0-3)
- **_miniProgramFilterIndex**: 当前选中的筛选标签索引 (0-2)
- **_showRegisterMiniProgramDialog**: 控制注册小程序弹窗显示
- **_showAddCompanyDialog**: 控制添加公司弹窗显示
- **_scrollController**: 控制页面滚动
- **监听AuthState变化**: 登录状态变化时刷新UI

### 9. 返回逻辑机制
- **从工作台返回逻辑**:
  - 从"我的"页面进入工作台 → 点击返回 → 返回到"我的"页面
  - 从首页进入工作台 → 点击返回 → 返回到首页
  - 从购买页进入工作台 → 点击返回 → 返回到购买页
  - 从租赁页进入工作台 → 点击返回 → 返回到租赁页
  - 从合作页进入工作台 → 点击返回 → 返回到合作页
  - 从定制开发页进入工作台 → 点击返回 → 返回到定制开发页
  - 直接访问工作台 → 点击返回 → 返回到"我的"页面
- **实现方式**:
  - 跳转时传递 from 参数: AppRouter.goToMerchantDashboard(context, from: AppRouter.profile)
  - 工作台通过 PopScope 拦截返回事件
  - 根据来源参数决定返回逻辑
