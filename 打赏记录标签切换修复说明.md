# 打赏记录标签切换修复说明

## 🐛 问题分析

用户反馈：点击"打赏记录"二级菜单时，不应该跳转到整个新页面，而应该在**同一页面内切换标签内容**。

之前的错误实现：
- ❌ 点击"打赏记录"会跳转到 `/workbench/dashboard/reward-records`
- ❌ 每次跳转都重新渲染整个页面（侧边栏、Header等）
- ❌ 用户体验不好，页面会闪烁

正确的实现：
- ✅ 点击"打赏记录"只是切换 `_selectedSubTab` 状态
- ✅ 根据状态在内容区显示不同的组件
- ✅ 侧边栏和Header保持不变，只切换内容区

## ✅ 修复内容

### 1. article_list_page.dart (文章列表页面)

**文件**: `lib/presentation/pages/workbench/article_management/article_list_page.dart`

#### 1.1 添加import
```dart
import 'reward_records_content.dart';
```

#### 1.2 修改二级菜单点击处理
修改 `_buildSecondLevelTabItem` 方法的 `onTap` 回调：

**修改前**:
```dart
onTap: () {
  // 点击二级菜单进行导航
  if (label == '打赏记录') {
    context.push('${AppRouter.merchantDashboard}/reward-records');
  }
}
```

**修改后**:
```dart
onTap: () {
  // 点击二级菜单切换标签内容
  setState(() {
    _selectedSubTab = label;
  });

  // 对于未实现的功能显示提示
  if (label == '分类管理') {
    ScaffoldMessenger.of(context).showSnackBar(
      const SnackBar(content: Text('分类管理页面即将推出')),
    );
  } else if (label == '付费管理') {
    ScaffoldMessenger.of(context).showSnackBar(
      const SnackBar(content: Text('付费管理页面即将推出')),
    );
  }
  debugPrint('切换到标签：$label');
}
```

#### 1.3 修改内容区显示逻辑
修改 `_buildMainContent()` 方法，根据标签显示不同内容：

**修改前**:
```dart
Widget _buildMainContent() {
  return Container(
    // ... 文章列表的固定内容
  );
}
```

**修改后**:
```dart
Widget _buildMainContent() {
  // 根据选中的二级标签显示不同的内容
  switch (_selectedSubTab) {
    case '打赏记录':
      return const RewardRecordsContent();
    case '文章列表':
      return _buildArticleListContent();
    case '分类管理':
    case '付费管理':
    default:
      return _buildArticleListContent();
  }
}

/// 文章列表内容区
Widget _buildArticleListContent() {
  return Container(
    // ... 原来的文章列表内容
  );
}
```
