# 讲师管理系统设计方案

## 一、功能概述

实现一个功能完善的讲师管理系统，包含：
- **总览页面**：表格形式展示所有讲师的核心数据
- **详情页面**：展示单个讲师的完整信息和数据分析
- **增删改查**：完整的讲师管理功能

## 二、数据结构设计

### 2.1 讲师数据模型

```dart
class Instructor {
  String id;
  String name;                    // 姓名
  String avatar;                  // 头像URL
  String bio;                     // 简介
  List<String> subjects;          // 学科标签
  DateTime registerTime;          // 注册时间
  InstructorStatus status;        // 登录状态
  CooperationMode cooperationMode;// 合作模式

  // 统计数据
  int studentCount;               // 学生数量
  int followerCount;              // 关注度
  double questionAnswerRate;      // 问答率 (0-1)
  double courseLikeRate;          // 课程点赞率 (0-1)
  double courseShareRate;         // 课程转发率 (0-1)
  double averageRating;           // 平均评分 (1-5)

  // 课程统计
  CourseStatistics courseStats;   // 课程统计详情
  RevenueStatistics revenueStats; // 收入统计

  // 预约与完结
  int reservedCourseCount;        // 预购课程数量
  int completedCourseCount;       // 已完结课程数量

  // 最近登录时间
  DateTime? lastLoginTime;

  // 学员评价
  List<StudentReview> reviews;
}

enum InstructorStatus {
  online,   // 在线
  busy,     // 忙碌
  offline   // 离线
}

enum CooperationMode {
  lease,    // 租赁
  partner   // 合作
}

class CourseStatistics {
  int singleCourses;      // 单课数量
  int seriesCourses;      // 套课数量
  int videoCourses;       // 视频数量
  int liveReplays;        // 直播回放数量
  int get total => singleCourses + seriesCourses + videoCourses + liveReplays;
}

class RevenueStatistics {
  double weeklyRevenue;   // 最近一周收入
  double monthlyRevenue;  // 最近一月收入
  double quarterlyRevenue;// 最近一季度收入
  double totalRevenue;    // 总收入
}

class StudentReview {
  String id;
  String studentName;
  String content;
  double rating;
  DateTime createTime;
}
```

### 3.2 表格列设计

| 列名 | 宽度 | 说明 |

| 头像 | 80px | 圆形头像，60x60 |
| 姓名 | 120px | 主要标识 |

| 课程数 | 100px | 总数，下方显示(单课:N 套课:N) |
| 收入 | 120px | 本月收入，如¥8,900 |
| 评分 | 100px | ⭐4.8，黄色星星 |
| 操作 | 150px | [详情][编辑][删除] 三个按钮 |

### 3.3 表格行设计（避免拥挤）

**每个讲师占3行**：
```
┌─────────────────────────────────────────────────────────┐
│ 👤  张三     计算机        🟢  合作  1,234  5,678       │

│    28门课程  ¥8,900本月  ⭐4.8      [详情][编辑][删除]  │

```

**第1行**：头像 + 姓名 + 学科 + 状态 + 模式 + 学生数 + 关注数
**第2行**：课程细分 + 收入 + 评分 + 操作按钮

## 七、UI设计要点

### 7.1 表格样式

- **宽度限制**：1200px，居中显示
- **表头**：深灰色背景 (#F5F6F7)，粗体文字
- **行高**：每行60px
- **边框**：1px浅灰色 (#E5E5E5)
- **悬停效果**：鼠标悬停时背景变为浅蓝色

### 7.2 状态指示

- 🟢 在线：绿色 (#52C41A)
- 🔴 忙碌：红色 (#FF4D4F)
- ⚪ 离线：灰色 (#D9D9D9)

### 7.3 数字格式化

- 大数字：1,234
- 收入：¥8,900、¥15k
- 百分比：92%、4.8/5.0

### 7.4 避免拥挤的技巧

1. **使用图标**：👤❤️📚💰⭐📊
2. **信息分层**：每个讲师3行，逐步展示
3. **合理间距**：行间距8px，列间距16px
4. **宽度控制**：每列有固定宽度
5. **标签折叠**：学科最多显示3个，超出显示"+N"

## 九、技术实现要点

### 9.1 性能优化

- 表格分页加载（每页10条）
- 图片懒加载
- 搜索防抖（500ms）

- 数字范围限制
