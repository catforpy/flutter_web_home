# 课程分类完整体系设计方案 v2.0

## 一、分类体系架构

课程分类管理包含**3个独立的分类维度**：

```
课程分类管理
├── 【分类类型】Tab切换
│   ├── 行业分类（已实现）
│   ├── 课程形式
│   └── 课程类型
```

## 二、三大分类维度

### 2.1 行业分类（Domain）
**说明**：按行业领域划分，最多4级

```
行业分类
├── 物流交通
│   ├── 仓储管理
│   │   ├── 入库流程
│   │   └── 出库流程
├── 科技培训
│   ├── 计算机
│   │   ├── Python
│   │   │   ├── 基础
│   │   │   └── 进阶
│   ├── 心理健康
│   └── 美食
├── 餐饮生活
├── 生活娱乐
└── 电商平台
```

### 2.2 课程形式分类（Format）
**说明**：按课程内容的表现形式划分

```
课程形式分类
├── 图文
│   ├── 纯文字
│   ├── 图文混排
│   └── 漫画
├── 音频
│   ├── 讲座
│   ├── 有声书
│   └── 播客
├── 视频
│   ├── 录播视频
│   ├── 直播回放
│   └── 动画演示
└── 直播
    ├── 视频直播
    ├── 音频直播
    └── 图文直播
```

### 2.3 课程类型分类（Type）
**说明**：按课程的售卖方式划分

```
课程类型分类
├── 单课
│   ├── 付费单课
│   ├── 免费单课
│   └── VIP专享
└── 套课
    ├── 系列课
    ├── 专题课
    ├── 训练营
    └── 年度会员
```

## 五、功能特性

### 5.1 分类类型管理
- ✅ 3个独立的分类类型
- ✅ Tab切换显示不同维度
- ✅ 每个类型独立的添加、编辑、删除
- ✅ 每个类型独立的默认数据

### 5.2 分类层级规则

| 分类类型 | 最大层级 | 说明 |
|---------|---------|------|
| 行业分类 | 4级 | 按行业细分 |
| 课程形式 | 2级 | 形式 > 具体类型（如视频 > 录播视频） |
| 课程类型 | 3级 | 类型 > 子类型 > 具体方式 |

### 5.3 分类操作
- 所有分类类型统一支持：
  - ✅ 添加分类（同级/子级）
  - ✅ 编辑名称
  - ✅ 删除分类（带确认）
  - ✅ 面包屑导航
  - ✅ 拖拽排序（待实现）

## 六、使用场景

### 6.1 课程发布时的分类选择

```
发布课程

├── 选择课程形式：视频 > 录播视频
├── 选择课程类型：单课 > 付费单课
└── 保存
```

### 6.2 用户筛选课程

```
课程列表
├── 筛选条件

│   ├── 形式：视频
│   └── 类型：套课
└── 显示符合条件的课程
```

## 八、界面交互优化

### 8.1 Tab切换动画
- 使用 `AnimatedContainer` 平滑切换
- 切换时显示对应的默认数据

### 8.2 分类标签样式
- 选中：主色背景 + 白色文字
- 普通：白色背景 + 深色文字
- 悬停：浅灰背景 + 阴影效果

### 8.3 操作反馈
- 添加成功 → SnackBar提示
- 编辑成功 → SnackBar提示
- 删除确认 → AlertDialog对话框
