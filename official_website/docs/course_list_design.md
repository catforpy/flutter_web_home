## 2. 数据模型设计

### 2.1 课程基本信息 (Course)

```dart
class Course {
  final String id;                    // 课程ID
  final String title;                 // 课程标题
  final String coverImage;            // 封面图片
  final String description;           // 课程简介

  // 分类信息
  final String? industryCategoryId;   // 行业分类ID
  final String? industryCategoryPath; // 行业分类路径: "计算机 > Python"
  final String? formatCategoryId;     // 形式分类ID: "single_course"
  final String? formatCategoryName;   // 形式分类名称: "单课"
  final String? typeCategoryId;       // 类型分类ID: "video_course"
  final String? typeCategoryName;     // 类型分类名称: "视频课"

  // 讲师信息
  final String instructorId;          // 讲师ID
  final String instructorName;        // 讲师姓名
  final String instructorAvatar;      // 讲师头像

  // 课程状态
  final CourseStatus status;          // 课程状态: presale/in_production/completed
  final CourseFormat format;          // 课程格式: single/series

  // 统计数据
  final int totalDuration;            // 总时长(分钟)
  final int lessonCount;              // 课时数(仅套课有)
  final int studentCount;             // 学员数
  final int likeCount;                // 点赞数
  final int reviewCount;              // 评价数
  final double averageRating;         // 平均评分

  // 价格信息
  final double price;                 // 价格
  final double originalPrice;         // 原价

  // 时间信息
  final DateTime createdAt;           // 创建时间
  final DateTime? publishedAt;        // 发布时间
}

enum CourseFormat {
  single('单课', 'single'),
  series('套课', 'series');

  final String label;
  final String value;
  const CourseFormat(this.label, this.value);
}
```

### 2.2 课程章节 (Lesson)

```dart
class Lesson {
  final String id;                    // 章节ID
  final String courseId;              // 课程ID
  final int chapterNumber;            // 章节序号
  final int lessonNumber;             // 课时序号
  final String title;                 // 课时标题
  final String outline;               // 课时大纲
  final int duration;                 // 时长(分钟)
  final bool isFree;                  // 是否免费试看
  final DateTime createdAt;           // 创建时间
  final int questionCount;            // 问答数量
}
```

### 2.3 课程章节分组 (Chapter)

```dart
class Chapter {
  final String id;                    // 章节ID
  final String courseId;              // 课程ID
  final int chapterNumber;            // 章节序号
  final String title;                 // 章节标题
  final List<Lesson> lessons;         // 课时列表
}
```

## 3. 核心功能

### 3.1 二级筛选功能

**筛选维度**：
1. **第一级：按讲师筛选**
   - 全部讲师
   - 具体讲师列表（动态加载）

2. **第二级：按分类筛选**
   - 全部分类
   - 具体行业分类列表（动态加载）

**筛选逻辑**：
- 可以同时选择"某个讲师" + "某个分类"
- 也可以只选择其中一项
- 默认显示"全部讲师" + "全部分类"

### 3.2 课程列表功能

**显示字段**：
- 课程封面
- 课程名称
- 分类信息（行业分类 | 形式 | 类型）
- 讲师名称
- 课程时长
- 课时数（仅套课显示）
- 学员数
- 点赞数
- 评价数
- 评分（星级显示）

**操作**：
- 查看详情
- 编辑
- 删除

### 3.3 课程详情功能

**三个Tab页**：

1. **课程简介**
   - 课程基本信息
   - 课程详细介绍
   - 讲师介绍卡片

2. **课程目录**
   - 按章节分组显示
   - 每节课显示：
     - 课时标题
     - 课时大纲
     - 时长
     - 问答数量
     - 查看问答按钮

3. **课程问答**
   - 复用课程问答管理中的课后问答组件
   - 显示该课程的所有课后问答
   - 支持提问和回答

## 4. API需求

### 4.1 课程列表接口

```
GET /api/courses

Query Parameters:
- page: int (页码)
- pageSize: int (每页数量)
- instructorId: string? (讲师ID，可选)
- industryCategoryId: string? (行业分类ID，可选)
- searchKeyword: string? (搜索关键词)

Response:
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 120,
    "courses": [Course]
  }
}
```

### 4.2 课程详情接口

```
GET /api/courses/{courseId}

Response:
{
  "code": 200,
  "message": "success",
  "data": {
    "course": Course,
    "chapters": [Chapter],
    "instructor": Instructor
  }
}
```

### 4.3 课时问答接口

```
GET /api/courses/{courseId}/lessons/{lessonId}/questions

Response:
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 12,
    "questions": [Question]
  }
}
```

## 5. 交互流程

### 5.1 查看课程详情流程

```
1. 用户在课程列表点击"查看详情"
2. 打开课程详情对话框/页面
3. 显示课程基本信息
4. 默认显示"课程简介"Tab
5. 用户可以切换到"课程目录"或"课程问答"Tab
6. 在课程目录中，可以点击"查看问答"查看某节课的问答
7. 在课程问答中，可以查看和回答问题
```

### 5.2 筛选流程

```
1. 用户点击讲师标签
2. 页面刷新显示该讲师的课程
3. 用户再点击分类标签
4. 页面刷新显示该讲师在该分类下的课程
5. 用户可以重置筛选条件
```
