## 🔐 后端V2接口路径

### 都达网账户（Platform Account）
```
POST /api/auth/v2/platform-account/register/password  // 注册
POST /api/auth/v2/platform-account/login/password     // 账号密码登录
POST /api/auth/v2/platform-account/login/sms          // 手机验证码登录
```

### 服务商（Service Provider）
```
POST /api/auth/v2/service-provider/register/password  // 注册
POST /api/auth/v2/service-provider/login/password     // 账号密码登录
POST /api/auth/v2/service-provider/login/sms          // 手机验证码登录
```

### 平台管理员（Platform Admin）
```
POST /api/auth/v2/platform-admin/register/password    // 注册
POST /api/auth/v2/platform-admin/login/password       // 账号密码登录
```

### 后台管理员（Backend Admin）
```
POST /api/auth/v2/backend-admin/register/password     // 注册
POST /api/auth/v2/backend-admin/login/password        // 账号密码登录
```

### 通用接口（所有身份共享）
```
POST /api/auth/v2/logout        // 登出
POST /api/auth/v2/refresh       // 刷新Token
GET  /api/auth/v2/validate      // 验证Token
GET  /api/auth/v2/user/info     // 获取用户信息
POST /api/auth/v2/sms/send      // 发送验证码
```

| `data.email` | string? | 邮箱 |

## 💻 Flutter 使用示例

### 1. 都达网账户登录

```dart
final authState = AuthState();

// 账号密码登录
final success = await authState.loginPlatformAccount(
  username: 'testuser',
  password: 'pass123456',
);

// 手机验证码登录
final success = await authState.loginPlatformAccountWithSms(
  phone: '13900139000',
  verifyCode: '123456',
);

if (success) {
  print('登录成功！');
  print('用户名: ${authState.username}');
  print('用户类型: ${authState.userType}');  // platform_account
  print('真实姓名: ${authState.realName}');
}
```

### 2. 服务商登录

```dart
// 账号密码登录
final success = await authState.loginServiceProvider(
  username: 'merchant001',
  password: 'pass123456',
);

// 手机验证码登录
final success = await authState.loginServiceProviderWithSms(
  phone: '13900139001',
  verifyCode: '123456',
);

if (success) {
  print('服务商登录成功！');
  print('用户类型: service_provider');
}
```

### 3. 平台管理员登录

```dart
final success = await authState.loginPlatformAdmin(
  username: 'admin001',
  password: 'admin123456',
);
```

### 4. 后台管理员登录

```dart
final success = await authState.loginBackendAdmin(
  username: 'backend001',
  password: 'backend123',
);
```

### 5. 发送验证码

```dart
final authServiceV2 = AuthServiceV2();
final result = await authServiceV2.sendSmsCode('13900139000');

result.fold(
  (failure) => print('发送失败: ${failure.message}'),
  (success) => print('验证码已发送'),
);
```

## 🚀 快速开始

### 示例：服务商账号密码登录

```dart
final authState = AuthState();

// 1. 发送验证码（可选，如果需要手机验证码登录）
final authServiceV2 = AuthServiceV2();
await authServiceV2.sendSmsCode('13900139001');

// 2. 账号密码登录
final success = await authState.loginServiceProvider(
  username: 'liuxizhuang',
  password: 'pass123456',
);

// 3. 检查登录状态
if (success) {
  print('✅ 登录成功');
  print('用户ID: ${authState.userId}');
  print('用户名: ${authState.username}');
  print('用户类型: ${authState.userType}');  // service_provider
  print('真实姓名: ${authState.realName}');
  print('账号状态: ${authState.status}');  // active
} else {
  print('❌ 登录失败: ${authState.errorMessage}');
}
```
