# Flutter Web - 后端认证V2接口对接指南

> **更新时间**: 2026-03-14
> **后端版本**: DudaNexus V2
> **基础路径**: `/api/auth/v2`

---

## 📋 V2接口设计原则

1. **明确区分不同身份**：每种身份有独立的登录/注册接口
2. **明确区分登录方式**：每种登录方式有独立的DTO
3. **支持同一人多身份叠加**

---

## 🎯 接口清单

### 一、都达网账户（Platform Account）

基础路径：`/api/auth/v2/platform-account`

#### 1. 账号密码注册
```
POST /api/auth/v2/platform-account/register/password
```

**请求体（PasswordRegisterReqDTO）**：
```json
{
  "username": "testuser",        // 必填
  "password": "pass123456",      // 必填
  "confirmPassword": "pass123456", // 必填
  "realName": "张三",            // 可选
  "phone": "13900139000"         // 可选
}
```

#### 2. 账号密码登录
```
POST /api/auth/v2/platform-account/login/password
```

**请求体（PasswordLoginReqDTO）**：
```json
{
  "username": "testuser",        // 必填
  "password": "pass123456"       // 必填
}
```

#### 3. 手机验证码登录
```
POST /api/auth/v2/platform-account/login/sms
```

**请求体（SmsCodeLoginReqDTO）**：
```json
{
  "phone": "13900139000",        // 必填
  "phoneVerifyCode": "123456"    // 必填
}
```

### 五、通用接口（所有身份共享）

基础路径：`/api/auth/v2`

#### 1. 用户登出
```
POST /api/auth/v2/logout
```

**请求参数**：
- Header: `Authorization: Bearer {accessToken}`
- Query: `userId={userId}`

#### 2. 刷新Token
```
POST /api/auth/v2/refresh
```

**请求参数**：
- Query: `refreshToken={refreshToken}`

#### 3. 验证Token
```
GET /api/auth/v2/validate
```

**请求参数**：
- Header: `Authorization: Bearer {accessToken}`

#### 4. 获取用户信息
```
GET /api/auth/v2/user/info
```

**请求参数**：
- Header: `Authorization: Bearer {accessToken}`

#### 5. 发送验证码
```
POST /api/auth/v2/sms/send
```

**请求参数**：
- Query: `phone={手机号}`

## 🔑 Token使用

### 登录成功后获取Token

登录成功后，Token会自动保存在本地：

```dart
final authState = AuthState();
await authState.loginPlatformAccount(
  username: 'testuser',
  password: 'pass123456',
);

// Token已自动保存，后续请求会自动携带
```

### 手动获取Token

```dart
final tokenManager = TokenManager();
final accessToken = await tokenManager.getAccessToken();
final refreshToken = await tokenManager.getRefreshToken();
```

### 使用Token调用其他接口

```dart
final apiClient = ApiClient();
final response = await apiClient.dio.get(
  '/api/some/endpoint',
  options: Options(
    headers: {
      'Authorization': 'Bearer $accessToken',
    },
  ),
);
```

## ⚠️ 注意事项

### 1. 用户类型（userType）

不同身份登录后，返回的 `userType` 不同：
- `normal` - 普通用户（都达网账户）
- `service_provider` - 服务商
- `platform_admin` - 平台管理员
- `backend_admin` - 后台管理员

### 2. Token过期处理

Token默认有效期为2小时（7200秒），过期后会自动刷新：

```dart
// ApiClient会自动处理401错误并刷新Token
final result = await apiClient.get('/some/endpoint');
```

### 3. 验证码有效期

短信验证码有效期为5分钟，请在此时间内完成登录。

### 4. 密码要求

- 最小长度：6位
- 建议包含：字母、数字、特殊字符

## 🚀 快速开始

### 1. 都达网账户注册并登录

```dart
// 注册
final authState = AuthState();
await authState.registerPlatformAccount(
  username: 'newuser',
  password: 'pass123456',
  confirmPassword: 'pass123456',
  realName: '新用户',
  phone: '13900139000',
);

// 如果注册成功，会自动登录
if (authState.isLoggedIn) {
  print('注册并登录成功！');
}
```

### 2. 使用已有账户登录

```dart
final authState = AuthState();
final success = await authState.loginPlatformAccount(
  username: 'testuser',
  password: 'pass123456',
);

if (success) {
  print('登录成功！');
  print('用户名: ${authState.username}');
  print('用户类型: ${authState.getUserTypeName()}');
}
```

### 3. 登出

```dart
await authState.logout();
print('已登出');
```
