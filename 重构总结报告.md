    return [...]; // 假数据集中管理
  }

// 通过 Repository 访问
class InstructorRepositoryImpl implements InstructorRepository {

    return Right(instructors);
  }
}

## 📐 新的架构设计

### 遵循 Clean Architecture

```
┌─────────────────────────────────────┐
│      Presentation Layer             │
│  (Pages, Widgets, UI)               │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│      Application Layer              │
│  (BLoCs, Events, States)            │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│         Domain Layer                │
│  (Entities, Repositories Interfaces)│
└──────────────┬──────────────────────┘
               ↑
┌──────────────┴──────────────────────┐
│          Data Layer                 │
│  (Models, Datasources, Repos Impl)  │
└─────────────────────────────────────┘
```

### 依赖关系

```
Presentation → Application → Domain
                            ↑
Data ────────────────────────┘
```

## 📖 最佳实践应用

### 1. SOLID 原则

**S - 单一职责**
```dart
class CarouselComponent extends PageComponent {
  // 只负责轮播组件的逻辑
}
```

**O - 开闭原则**
```dart
abstract class PageComponent {
  Widget buildPreview(BuildContext context);
}

// 新增组件只需继承基类，无需修改现有代码
class NewComponent extends PageComponent {
  @override
  Widget buildPreview(BuildContext context) { /* ... */ }
}
```

**L - 里氏替换**
```dart
List<PageComponent> components = [
  CarouselComponent(),
  VideoComponent(),
  NoticeComponent(),
]; // 所有组件可互换
```

**I - 接口隔离**
```dart
abstract class PageComponent {
  Widget buildPreview(BuildContext context);
  Widget buildEditor(BuildContext context);
} // 接口精简，职责明确
```

**D - 依赖倒置**
```dart
class InstructorBloc {
  final InstructorRepository repository; // 依赖抽象接口

  InstructorBloc({required this.repository});
}
```

### 2. DRY 原则

**提取公共基类**
```dart
abstract class ComponentStyle {
  final EdgeInsets margin;
  final EdgeInsets padding;
  final double borderRadius;

  // 公共逻辑统一实现
}
```

**提取公共方法**
```dart
static Color? parseColor(String? colorString) {
  if (colorString == null || colorString.isEmpty) return null;
  return Color(int.parse(colorString.replaceFirst('#', '0xFF')));
}

// 所有组件都使用这个方法
```

### 3. 命名规范

```dart
// 文件名：snake_case
instructor_management_page.dart
instructor_filter_panel.dart

// 类名：PascalCase
class InstructorManagementPage {}
class InstructorFilterPanel {}

// 变量/方法：camelCase
final String searchKeyword;
void loadInstructors() {}

// 常量：lowerCamelCase
const defaultStyle = CarouselStyle();

// 私有成员：前缀 _
class _InstructorManagementView {}
void _handleClick() {}
```
