# 打赏记录页面使用指南

## 📋 页面概述

**页面名称**: 打赏记录 (Reward Records)
**所属模块**: 资讯管理 - 文章管理
**路由路径**: `/workbench/dashboard/reward-records`
**文件位置**:
- 主页面: `lib/presentation/pages/workbench/article_management/reward_records_page.dart`
- 内容区: `lib/presentation/pages/workbench/article_management/reward_records_content.dart`

---

## 🎯 页面功能

### 1. 搜索筛选区

提供以下筛选条件：

| 筛选条件 | 说明 | 占位符 |
|---------|------|--------|
| 打赏会员 | 支持模糊搜索会员昵称 | 请输入会员昵称 |
| 文章标题 | 支持模糊搜索文章标题 | 请输入文章标题 |
| 打赏时间 | 日期范围选择（开始时间 - 截止时间） | 开始时间 / 截止时间 |

**操作按钮**:
- **查询** - 根据筛选条件查询数据
- **重置** - 清空所有筛选条件
- **导出Excel** - 导出当前查询结果（功能开发中）

### 2. 数据表格

展示打赏流水明细，包含以下列：

| 列名 | 说明 | 对齐方式 | 特殊处理 |
|------|------|----------|----------|
| 订单编号 | 打赏订单号 | 左对齐 | 长数字串 |
| 文章标题 | 被打赏的文章名称 | 左对齐 | 支持换行，最多显示2行 |
| 打赏会员 | 打赏者昵称 | 左对齐 | - |
| 打赏金额 | 打赏金额数值 | 右对齐 | 橙色突出显示 (¥)，保留2位小数 |
| 类型 | 固定显示"资讯" | 居中对齐 | - |
| 支付方式 | 如"余额支付" | 居中对齐 | - |
| 打赏时间 | 格式: YYYY-MM-DD HH:mm | 右对齐 | - |

**表格特性**:
- ✅ 斑马纹显示（偶数行浅灰色背景）
- ✅ 鼠标悬停高亮效果
- ✅ 长标题自动换行（最多2行）
- ✅ 金额橙色突出显示
- ✅ 空状态友好提示

### 3. 分页器

- 显示总记录数和每页条数
- 支持上一页/下一页导航
- 页码快速跳转
- 智能页码显示（当前页前后优先显示）

## 🔧 代码结构

### 主页面 (reward_records_page.dart)

- 左侧一级导航栏（200px）
- 二级标签栏（120px）
- 顶部Header（60px）


### 内容区 (reward_records_content.dart)

包含核心业务逻辑：
- 搜索筛选区
- 数据表格

```dart
RewardRecordsContent
├── _buildSearchArea()        // 搜索筛选区
```

## 🐛 常见问题

### Q1: 如何修改每页显示的条数？

A: 在 `reward_records_content.dart` 中修改 `_pageSize` 的值：

```dart
final int _pageSize = 20; // 改为20条/页
```

### Q2: 如何添加更多筛选条件？

A: 在 `_buildSearchArea()` 方法中添加新的筛选控件，并在 `_filteredRecords` getter 中添加相应的过滤逻辑。

### Q3: 如何修改表格列的宽度？

A: 在 `_buildTableHeader()` 和 `_buildTableRow()` 方法中调整每个列的 `flex` 值或固定宽度。
