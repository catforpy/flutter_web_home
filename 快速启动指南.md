# 🚀 快速启动指南

## 立即运行（3个命令）

### 步骤1: 进入项目目录
```bash
cd "/Volumes/DudaDate/微信小程序开发/flutter官网开发/official_website"
```

### 步骤2: 启动应用
```bash
flutter run -d chrome --web-port 8080
```

### 步骤3: 外网访问（可选）

**另开一个终端窗口**，运行：

```bash
# 安装ngrok（如果还没安装）
brew install ngrok

# 启动ngrok
ngrok http 8080

# 复制生成的地址，例如: https://xxx.ngrok.io
```

## 🎮 热重载技巧

应用运行后，修改代码：

- 按 **`r`** 键 → 热重载（保留状态）
- 按 **`R`** 键 → 热重启（重置状态）
- 按 **`q`** 键 → 退出
- 按 **`h`** 键 → 查看帮助

## 🔧 常用操作

### 修改代码并查看效果
1. 编辑任意 `.dart` 文件
2. 保存文件
3. 在终端按 `r` 键

flutter clean

## 🌐 分享你的网站

### 临时分享（使用ngrok）

```bash
# 1. 确保应用正在运行
flutter run -d chrome --web-port 8080

# 2. 另开一个终端，运行ngrok
ngrok http 8080

# 3. 复制生成的Forwarding地址
# 例如: https://abc123.ngrok.io

# 4. 分享这个地址给任何人！
```

### 永久部署（使用Vercel）

```bash
# 1. 构建项目
flutter build web --release

# 2. 安装Vercel CLI
npm i -g vercel

# 3. 部署
cd build/web
vercel

# 4. 获得永久域名
# 例如: https://your-site.vercel.app
```

## ❓ 遇到问题？

### 应用无法启动
```bash
# 清理并重新构建
flutter clean
flutter pub get
flutter run -d chrome
```

### 端口被占用
```bash
# 使用其他端口
flutter run -d chrome --web-port 3000
```

### Chrome无法打开
```bash
# 列出可用设备
flutter devices

# 使用指定设备
flutter run -d chrome --device-id <device-id>
```
