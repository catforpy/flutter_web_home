### 2. ✅ **HTTP请求拦截器 - 自动附加Token**
**实现位置**：
- `lib/core/network/api_client.dart`（第60-77行）

**功能说明**：
- ✅ 每次发送API请求前，自动从TokenManager获取access token
- ✅ 自动将token附加到请求头：`Authorization: Bearer {token}`
- ✅ 如果token为空，不附加Authorization头
- ✅ 支持网络日志，方便调试

**代码实现**：
```dart
_dio.interceptors.add(InterceptorsWrapper(
  onRequest: (options, handler) async {
    // 从TokenManager获取 Token
    final token = await _tokenManager.getAccessToken();
    if (token != null && token.isNotEmpty) {
      options.headers['Authorization'] = 'Bearer $token';
    }
    handler.next(options);
  },
));
```

### 3. ✅ **Token自动刷新机制**
**实现位置**：
- `lib/core/network/api_client.dart`（第85-137行）
- `lib/core/services/auth_v2_service.dart`（第233-278行）

**功能说明**：
- ✅ 检测到401错误时，自动尝试刷新token
- ✅ 使用refresh token调用后端刷新接口：`POST /api/auth/v2/refresh`
- ✅ 带锁机制，防止并发刷新（_isRefreshing）
- ✅ 请求队列机制，多个401请求只刷新一次token
- ✅ 刷新成功后，自动重试所有失败的请求
- ✅ 使用V2接口：`AuthServiceV2.refreshToken()`

**代码实现**：
```dart
// 检测401错误
onError: (error, handler) async {
  if (error.response?.statusCode == 401) {
    if (_isRefreshing) {
      // 正在刷新，请求加入队列
      _retryRequests.add(_RetryRequest(error.requestOptions, handler));
      return;
    }

    // 开始刷新Token
    _isRefreshing = true;
    final refreshed = await _refreshTokenAndRetry();
    _isRefreshing = false;

    if (refreshed) {
      // 刷新成功，重试原请求和队列中的请求
      final response = await _retry(error.requestOptions);
      // 重试队列中的请求...
      return handler.resolve(response);
    }
  }
  handler.next(error);
}
```

**代码实现**：
```dart
void _handleRefreshFailure() {
  _logger.w('⚠️  Token刷新失败，清除本地Token并准备跳转登录页');
  // 清除本地Token
  _tokenManager.clearTokens();

  // 跳转到登录页面
  Future.microtask(() {
    final context = navigatorKey.currentContext;
    if (context != null) {
      Navigator.of(context).pushNamedAndRemoveUntil(
        '/',
        (route) => false, // 清除所有路由
      );
    }
  });
}
```

### 6. ✅ **应用启动时自动恢复登录状态**
**实现位置**：
- `lib/main.dart`
- `lib/core/auth/auth_state.dart`（第137-162行）

**功能说明**：
- ✅ 应用启动时初始化StorageService
- ✅ 应用启动时调用`AuthState.initialize()`
- ✅ 自动检查本地是否有有效token
- ✅ 如果token有效，自动获取用户信息并恢复登录状态
- ✅ 如果token无效，清除登录状态

**代码实现**：
```dart
// main.dart
void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await StorageService().init();  // 初始化存储服务
  runApp(const OfficialWebsiteApp());
}

// AuthState.initialize()
Future<void> initialize() async {
  final hasToken = await _tokenManager.hasValidToken();
  if (hasToken) {
    // 有Token，尝试获取用户信息
    final result = await _authService.getUserInfo();
    result.fold(
      (failure) {
        // Token无效，清除登录状态
        _clearAuthState();
      },
      (userInfo) {
        // Token有效，恢复登录状态
        _userInfo = userInfo;
        _isLoggedIn = true;
        notifyListeners();
      },
    );
  }
}
```
