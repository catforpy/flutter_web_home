## 🏗️ 架构说明

### 目录结构

```
lib/
├── domain/                    # 领域层 - 业务实体和接口
│   ├── entities/             # 实体类
│   │   ├── page_component.dart
│   │   └── components/       # 具体组件实体
│   └── repositories/         # 仓储接口
│
├── data/                     # 数据层 - 数据实现
│   ├── datasources/          # 数据源（包括假数据）
│   ├── models/               # 数据模型
│   └── repositories/         # 仓储实现
│
├── application/              # 应用层 - 业务逻辑
│   └── blocs/               # BLoC 状态管理
│       ├── feature_name/     # 按功能分类
│       │   ├── feature_bloc.dart
│       │   ├── feature_event.dart
│       │   └── feature_state.dart
│
└── presentation/             # 表现层 - UI 组件
    └── pages/
        └── feature_name/
            ├── feature_page.dart       # 主页面 < 300 行
            └── widgets/               # 页面组件
                ├── widget_1.dart
                └── widget_2.dart
```

### 依赖规则

```
Presentation → Application → Domain
                             ↑
Data ────────────────────────┘
```

**关键原则**:
- ✅ 外层依赖内层
- ✅ 内层不依赖外层
- ✅ Domain 层完全独立

**2. State**

**文件**: `application/blocs/my_entity/my_entity_state.dart`

```dart
import 'package:equatable/equatable.dart';
import 'package:official_website/domain/entities/my_entity.dart';

abstract class MyEntityState extends Equatable {
  const MyEntityState();

  @override
  List<Object?> get props => [];
}

class MyEntityInitial extends MyEntityState {
  const MyEntityInitial();
}

class MyEntityLoading extends MyEntityState {
  const MyEntityLoading();
}

class MyEntityLoaded extends MyEntityState {
  final List<MyEntity> entities;
  final int currentPage;
  final int totalCount;
  final bool hasMore;
  final String? searchKeyword;

  const MyEntityLoaded({
    this.entities = const [],
    this.currentPage = 1,
    this.totalCount = 0,
    this.hasMore = false,
    this.searchKeyword,
  });

  @override
  List<Object?> get props => [
        entities,
        currentPage,
        totalCount,
        hasMore,
        searchKeyword,
      ];

  MyEntityLoaded copyWith({
    List<MyEntity>? entities,
    int? currentPage,
    int? totalCount,
    bool? hasMore,
    String? searchKeyword,
  }) {
    return MyEntityLoaded(
      entities: entities ?? this.entities,
      currentPage: currentPage ?? this.currentPage,
      totalCount: totalCount ?? this.totalCount,
      hasMore: hasMore ?? this.hasMore,
      searchKeyword: searchKeyword ?? this.searchKeyword,
    );
  }
}

class MyEntityError extends MyEntityState {
  final String message;

  const MyEntityError(this.message);

  @override
  List<Object?> get props => [message];
}
```

**3. BLoC**

**文件**: `application/blocs/my_entity/my_entity_bloc.dart`

```dart
import 'package:flutter_bloc/flutter_bloc.dart';
import 'package:official_website/application/blocs/my_entity/my_entity_event.dart';
import 'package:official_website/application/blocs/my_entity/my_entity_state.dart';
import 'package:official_website/domain/repositories/my_entity_repository.dart';

class MyEntityBloc extends Bloc<MyEntityEvent, MyEntityState> {
  final MyEntityRepository repository;

  MyEntityBloc({required this.repository}) : super(const MyEntityInitial()) {
    on<LoadEntitiesEvent>(_onLoadEntities);
    on<SearchEntitiesEvent>(_onSearchEntities);
    on<DeleteEntityEvent>(_onDeleteEntity);
  }

  Future<void> _onLoadEntities(
    LoadEntitiesEvent event,
    Emitter<MyEntityState> emit,
  ) async {
    emit(const MyEntityLoading());

    final result = await repository.getEntities(
      page: event.page,
      pageSize: event.pageSize,
      searchKeyword: event.searchKeyword,
    );

    result.fold(
      (failure) => emit(MyEntityError(failure.message)),
      (entities) {
        emit(MyEntityLoaded(
          entities: entities,
          currentPage: event.page,
          totalCount: entities.length,
          hasMore: entities.length == event.pageSize,
          searchKeyword: event.searchKeyword,
        ));
      },
    );
  }

  Future<void> _onSearchEntities(
    SearchEntitiesEvent event,
    Emitter<MyEntityState> emit,
  ) async {
    if (state is MyEntityLoaded) {
      final currentState = state as MyEntityLoaded;

      emit(const MyEntityLoading());

      final result = await repository.searchEntities(event.keyword);

      result.fold(
        (failure) => emit(MyEntityError(failure.message)),
        (entities) {
          emit(currentState.copyWith(
            entities: entities,
            currentPage: 1,
            searchKeyword: event.keyword,
          ));
        },
      );
    }
  }

  Future<void> _onDeleteEntity(
    DeleteEntityEvent event,
    Emitter<MyEntityState> emit,
  ) async {
    final result = await repository.deleteEntity(event.id);

    result.fold(
      (failure) => emit(MyEntityError(failure.message)),
      (_) {
        if (state is MyEntityLoaded) {
          final currentState = state as MyEntityLoaded;
          add(LoadEntitiesEvent(
            page: currentState.currentPage,
            pageSize: 10,
          ));
        }
      },
    );
  }
}
```

```dart
import 'package:flutter/material.dart';

class FeaturePage extends StatelessWidget {
  const FeaturePage({super.key});

  @override
  Widget build(BuildContext context) {
    return BlocProvider(
      create: (context) => MyEntityBloc(
        repository: MyEntityRepositoryImpl(),
      )..add(const LoadEntitiesEvent(page: 1, pageSize: 10)),
      child: const _FeaturePageView(),
    );
  }
}

class _FeaturePageView extends StatelessWidget {
  const _FeaturePageView();

    return Scaffold(
      appBar: AppBar(
        title: const Text('功能页面'),
        actions: [
          IconButton(
            icon: const Icon(Icons.refresh),
            onPressed: () {
              context.read<MyEntityBloc>().add(
                    const LoadEntitiesEvent(page: 1, pageSize: 10),
                  );
            },
          ),
        ],
      ),
      body: const EntityListView(),
    );
  }
}
```

**2. 列表视图**

**文件**: `presentation/pages/feature_name/widgets/entity_list_view.dart`

```dart
import 'package:flutter/material.dart';
import 'package:flutter_bloc/flutter_bloc.dart';
import 'package:official_website/application/blocs/my_entity/my_entity_bloc.dart';
import 'package:official_website/application/blocs/my_entity/my_entity_state.dart';

class EntityListView extends StatelessWidget {
  const EntityListView({super.key});

  @override
  Widget build(BuildContext context) {
    return BlocBuilder<MyEntityBloc, MyEntityState>(
      builder: (context, state) {
        if (state is MyEntityLoading) {
          return const Center(child: CircularProgressIndicator());
        }

        if (state is MyEntityError) {
          return Center(child: Text('错误: ${state.message}'));
        }

        if (state is MyEntityLoaded) {
          if (state.entities.isEmpty) {
            return const Center(child: Text('暂无数据'));
          }

          return ListView.builder(
            padding: const EdgeInsets.all(16),
            itemCount: state.entities.length,
            itemBuilder: (context, index) {
              final entity = state.entities[index];
              return EntityCard(entity: entity);
            },
          );
        }

        return const SizedBox.shrink();
      },
    );
  }
}
```

| 文件类型 | 推荐行数 | 最大行数 |

| 页面主文件 | < 200 | 300 |

### 2. 命名规范

```dart
// ✅ 文件名：snake_case
instructor_management_page.dart

// ✅ 类名：PascalCase

// ✅ 变量/方法：camelCase

void loadInstructors() {}

// ✅ 常量：lowerCamelCase
const defaultStyle = CarouselStyle();

// ✅ 私有成员：前缀 _

void _handleClick() {}
```

## ❓ 常见问题

### Q1: 如何判断一个文件需要重构？

**指标**:
- 文件超过 500 行
- 类超过 300 行
- 方法超过 50 行
- 硬编码大量数据
- 缺乏状态管理

### Q2: 如何拆分大文件？

**步骤**:
1. 提取实体类到 `domain/entities/`
2. 提取数据到 `data/datasources/`
3. 创建 BLoC 到 `application/blocs/`
4. 拆分 UI 组件到 `widgets/`
5. 重构主页面文件

### Q3: 假数据应该放在哪里？

**位置**: `data/datasources/feature_mock_datasource.dart`

**原则**:
- 所有假数据集中管理
- 命名规范：`{Feature}MockDatasource`
- 方便后续替换为真实 API

### Q4: 如何创建新的 BLoC？

**模板**:
```
application/blocs/feature_name/
├── feature_bloc.dart     - BLoC 逻辑
├── feature_event.dart    - 事件定义
└── feature_state.dart    - 状态定义
```

### Q5: 如何添加新的组件类型？

**步骤**:
1. 在 `domain/entities/components/` 创建组件实体
2. 在 `ComponentType` 枚举中添加类型
3. 在 `PageComponent.fromJson` 中添加解析逻辑
4. 创建对应的预览和编辑器组件
