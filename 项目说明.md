### 2️⃣ 页面之间怎么跳转？

**答案：使用 go_router 进行路由管理**

**路由配置文件**: `lib/presentation/routes/app_router.dart`

**核心代码**：
```dart
class AppRouter {
  static const String home = '/';
  static const String about = '/about';
  static const String contact = '/contact';

  static final GoRouter router = GoRouter(
    initialLocation: home,
    routes: [
      GoRoute(path: home, builder: (_, __) => const HomePage()),
      GoRoute(path: about, builder: (_, __) => const AboutPage()),
      GoRoute(path: contact, builder: (_, __) => const ContactPage()),
    ],
  );
}
```

**使用方式**：
```dart
// 方式1: 使用静态方法
AppRouter.goToHome(context);    // 去首页
AppRouter.goToAbout(context);   // 去关于页面
AppRouter.goToContact(context); // 去联系页面

// 方式2: 直接使用go_router
context.go('/about');      // 导航到关于页面
context.push('/contact');   // push到联系页面
context.pop();              // 返回上一页
```

**特点**：
- ✅ 声明式配置
- ✅ 类型安全
- ✅ 深链接支持
- ✅ URL自动同步
- ✅ 浏览器前进后退支持

### 3️⃣ 有状态管理吗？用什么？

**答案：有！使用 BLoC (flutter_bloc)**

**状态管理文件位置**: `lib/application/blocs/`

**已实现的BLoC**：

1. **CounterBloc** - 计数器状态管理
2. **ThemeBloc** - 主题切换状态管理

**BLoC工作原理**：
```
事件(Event) → BLoC处理 → 新状态(State) → UI更新
```

**完整示例**：

```dart
// 1. 定义事件
abstract class CounterEvent extends Equatable {
  const CounterEvent();
  @override
  List<Object> get props => [];
}

class IncrementEvent extends CounterEvent {}
class DecrementEvent extends CounterEvent {}

// 2. 定义状态
class CounterState extends Equatable {
  final int count;
  const CounterState({this.count = 0});

  @override
  List<Object> get props => [count];
}

// 3. 实现BLoC
class CounterBloc extends Bloc<CounterEvent, CounterState> {
  CounterBloc() : super(const CounterState()) {
    on<IncrementEvent>((event, emit) {
      emit(CounterState(count: state.count + 1));
    });
    on<DecrementEvent>((event, emit) {
      emit(CounterState(count: state.count - 1));
    });
  }
}

// 4. 在UI中使用
BlocBuilder<CounterBloc, CounterState>(
  builder: (context, state) {
    return Text('计数: ${state.count}');
  },
)

// 5. 触发事件
ElevatedButton(
  onPressed: () {
    context.read<CounterBloc>().add(IncrementEvent());
  },
  child: Text('增加'),
)
```

**在应用中提供BLoC**：
```dart
// main.dart
MultiBlocProvider(
  providers: [
    BlocProvider(create: (_) => CounterBloc()),
    BlocProvider(create: (_) => ThemeBloc()),
  ],
  child: MaterialApp.router(...),
)
```

### 4️⃣ 如何通过外网访问？

**答案：最简单使用 ngrok**

#### 方法1: ngrok（临时分享，推荐）

```bash
# 1. 安装ngrok
brew install ngrok

# 2. 启动Flutter应用（终端1）
cd "/Volumes/DudaDate/微信小程序开发/flutter官网开发/official_website"
flutter run -d chrome --web-port 8080

# 3. 启动ngrok（终端2）
ngrok http 8080

# 4. 复制生成的地址
# 例如: https://abc123.ngrok.io

# 5. 分享这个地址给任何人！
```

**优点**：
- ✅ 免费
- ✅ 无需配置
- ✅ HTTPS支持
- ✅ 即时生效

#### 方法2: Cloudflare Tunnel（推荐）

```bash
# 安装
brew install cloudflare/cloudflare/cloudflared

# 使用
cloudflared tunnel --url http://localhost:8080
```

#### 方法3: Vercel/Netlify（永久部署）

```bash
# 1. 构建项目
flutter build web --release

# 2. 安装Vercel CLI
npm i -g vercel

# 3. 部署
cd build/web
vercel

# 4. 获得永久域名
# https://your-site.vercel.app
```

#### 方法4: GitHub Pages（免费托管）

```bash
# 1. 构建项目
flutter build web --release

# 2. 推送到GitHub
git init
git add .
git commit -m "Deploy"
git push origin main

# 3. 在GitHub设置中启用Pages
# Settings > Pages > Source: main branch
```

### 添加新页面

**步骤1**: 创建页面文件
```dart
// lib/presentation/pages/mynewpage/my_new_page.dart
class MyNewPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(child: Text('新页面')),
    );
  }
}
```

**步骤2**: 在路由中注册
```dart
// lib/presentation/routes/app_router.dart
GoRoute(
  path: '/mynewpage',
  builder: (context, state) => const MyNewPage(),
)
```

**步骤3**: 使用
```dart
AppRouter.goToMyNewPage(context); // 或 context.go('/mynewpage')
```
